# 快速创建talentui模版项目

1. 安装 
```
sudo yarn global add @talentui/cli 
```

2. 执行创建项目

```
//talent init 目标文件夹名称(如果为 . 则在当前目录下生成项目)
talent init demo
```
3. 选择vue／react模版
4. 根据提示完善项目信息
5. 完成创建
